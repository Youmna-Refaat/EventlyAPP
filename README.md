# Evently App

## Overview

Evently is a mobile app designed to help users discover and connect with events happening around them. With a user-friendly interface and a robust feature set, Evently aims to make it easy for people to find and attend events that match their interests.

## Features

* **Event Discovery:** Browse through a curated list of events happening in your area, filtered by category, location, and date.
* **Event Details:** View detailed information about each event, including description, location, date, time, and ticketing information.
* **Personalized Recommendations:** Get personalized event recommendations based on your interests and preferences.
* **Event Creation:** Create and manage your own events, including setting up event details, ticketing, and promotion.
* **Social Sharing:** Share events with friends and family on social media platforms.
* **Maps Integration:** View event locations on a map and get directions to the venue.
* **Push Notifications:** Receive reminders and updates about upcoming events.

## Technical Details

* **Built with Flutter:** Evently is built using the Flutter framework, ensuring a seamless and consistent user experience across both iOS and Android platforms.
* **Firebase Integration:** Evently uses Firebase for authentication, real-time database, and cloud functions.
* **Google Maps Integration:** Evently uses the Google Maps API for maps and directions.

## Skills

* **Event Management:** Evently allows users to create, manage, and promote their own events.
* **Event Discovery:** Evently's algorithm-driven event discovery feature helps users find events that match their interests.
* **Social Sharing:** Evently enables users to share events on social media platforms.

## How it Works

1.  Users download and install the Evently app.
2.  Users create an account or log in using their existing credentials.
3.  Users browse through the event list, filtered by category, location, and date.
4.  Users view event details, including description, location, date, time, and ticketing information.
5.  Users can create and manage their own events, including setting up event details, ticketing, and promotion.
6.  Users receive personalized event recommendations based on their interests and preferences.

## Installation Steps

### Prerequisites

* Flutter SDK installed on your machine.
* Firebase account set up.
* Google Maps API key set up.

### Step 1: Clone the Repository

```bash
git clone [https://github.com/your-username/evently.git](https://github.com/your-username/evently.git)
```
### Step 2: Install Dependencies

```bash
flutter pub get
```
### Step 3: Set up Firebase

# Set up Firebase by creating a new project in the Firebase console and enabling the Firebase Realtime Database and Firebase Authentication.
# Firebase documentation for detailed setup instructions: [https://firebase.google.com/docs/build]

### Step 4: Set up Google Maps API

# Set up the Google Maps API by creating a new project in the Google Cloud Console and enabling the Google Maps JavaScript API.
#  Google Cloud documentation for detailed setup instructions: [https://cloud.google.com/docs/]

### Step 5: Build and Run the App

```bash
flutter run
```
